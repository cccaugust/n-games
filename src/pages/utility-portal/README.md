# ユーティリティ ポータル（N-Utilities）

## 概要
ゲームとは別の「ユーティリティ系アプリ一覧」ページです。  
ログイン（あいことば: `0818`）とユーザー選択（プレイヤー）はゲーム側と共通です。

## 使い方（プレイヤー視点）
- 画面に入る → あいことばを数字ボタンで入力 → ユーザーを選ぶ
- ユーティリティカードを押すと各アプリに移動します

## 構成
- `portal.html`: ユーティリティ一覧ページ（ミニマルデザイン）
- `portal.js`: 一覧の描画とヘッダー操作（交代/ログアウト/ゲームへ）
- `apps.js`: 表示するユーティリティ一覧（ここに追加して増やす）

## 追加方法（開発者向け）
`apps.js` の `utilityApps` に1件追加します。

例:

```js
export const utilityApps = [
  // ...
  {
    id: 'my-tool',
    title: 'My Tool',
    desc: '説明',
    icon: '🛠️',
    link: '/pages/my-tool/'
  }
];
```

## メモ
認証は `src/js/auth.js` の `requireAuth()` を使っています。  
未ログイン/未選択の場合はログイン・ユーザー選択へ飛び、完了後は元のページへ戻る仕組みです。
