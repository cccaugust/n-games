# Page: mon-survivor

## 概要
大量の敵が迫ってくる中で、できるだけ長く生き残るサバイバー系アクションです（攻撃はオート）。

## 操作方法
- **PC**: 矢印キー / WASD で移動
- **タッチ**: 画面左下の仮想スティックで移動（スマホ/タブレット）

## ルール / 仕様
- **目的**: 敵に接触して減るHPが 0 になる前に、できるだけ長く生き残る
- **スコア**: 倒した敵の数（画面右上のゲームオーバー表示で確認）
- **ゲームオーバー**: HP が 0 になると終了

## 画面 / UI
- 上部: レベル / 経験値バー / 経過時間
- 下部: HPバー
- レベルアップ時: 強化カード選択（停止中）
- スマホ/タブレット: 左下に仮想スティック表示

## 実装メモ（設計）
- **エントリ**: `index.html`
- **メインロジック**: `game.js`
- **主要な状態(state)**: player, enemies, projectiles, particles, gems, frame, score, isGameOver, isPaused
- **データ保存**: なし

## ファイル構成
- `index.html`: 画面レイアウト（canvas + HUD + 画面UI）
- `style.css`: 見た目（HUD/モーダル/仮想スティック含む）
- `game.js`: メインループ、スポーン、当たり判定、UI更新、入力（キーボード + 仮想スティック）
- `entities.js`: Player/Enemy/Projectile などのクラス
