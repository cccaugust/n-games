布のパッチを買って、9×9のキルトを綺麗に埋めよう！時間とボタン（お金）の使いどころが勝敗を分ける、戦略タイル配置ゲーム。

2人専用 / ボタン5個スタート / 時間トラック25

あそびかた

1後ろにいる人の番

時間トラックで 後ろにいる人 が常に手番。同じマスにいるときは、後から来た人が先に動く。

2パッチを買う or 進む

毎ターン、次の2つから1つ選ぶ:
A. 次に並んでる3枚から1枚を買う → 🪙コスト払って → ⏰時間進めて → 9×9キルトに置く
B. 「進む」 相手の1マス先まで進んで、進んだマス数だけ🪙ボタンをもらう

3パッチを置く

選んだパッチは ↻回転 / ⇋反転 できる。キルトの空きマスをタップして配置！キーボード R / F / Esc も使える。

4ボタン収入

パッチには ●ボタン印 がついてるものがあり、その合計が「収入」。時間トラックの 🪙マスを通過するたびに、収入分のボタンをもらえる！

5勝敗

両者がゴール🏁に着いたら終了。
スコア = 持ってる🪙 − 空きマス×2
スコアが多い人の勝ち！
