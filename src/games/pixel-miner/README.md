# Game: pixel-miner（ピクセルマイナー）

## 概要
2Dの「マイクラ/テラリアっぽい」ミニゲームです。地形を掘ってブロックを集め、好きなところに置いて遊べます。

このゲームは **ドット絵メーカーのサンプル素材**（タイル/アイテム/キャラ）を読み込んで表示しています。

## 操作方法
- **PC**
  - 移動: `A/D` または `←/→`
  - ジャンプ: `W/↑/Space`
  - 掘る: 左クリック（押しっぱなしOK）
  - 置く: 右クリック、または `E` で置くモード
  - 掘るモード: `Q`
  - ホットバー: `1`〜`5`
  - ズーム: `+` / `-`

- **タッチ**
  - 画面下のボタンで移動/ジャンプ
  - 「⛏ / 🧱」でモード切り替え
  - キャンバスをタップで掘る/置く

## ルール / 仕様（いまの最小版）
- **ワールド**: 256×128 タイル、簡易地形生成 + 洞窟
- **ブロック**: くさつち / つち / いし / すな / もくざい / レンガ / みず / ようがん など
- **キャラ**: プレイヤー/モンスターは **2フレームの簡易アニメ**（ドット絵メーカーの `frames` を使用）
- **演出**: 採掘/破壊/設置/ジャンプ/被弾の効果音、破壊時の粉塵パーティクル、軽い画面揺れ
- **サウンド**: 右上の「🔊/🔇」でON/OFF（設定はブラウザに保存）
- **設定**: 右上の「⚙」から、プレイヤーの見た目を **サンプル / 自作（ドット絵メーカー）** で変更できます
- **保存**: `localStorage` に自動保存（ページ更新しても続きから）

## ファイル構成
- `index.html`: 画面・UI
- `settings.html`: 設定（プレイヤースキン選択）
- `style.css`: 見た目
- `settings.css`: 設定画面の見た目
- `game.js`: ゲーム本体（生成/物理/描画/採掘/設置/保存）
- `settings.js`: 設定画面ロジック（サンプル/自作一覧、保存）
