# クアルト！（Quarto）

2人専用の名作アブストラクトボードゲーム。1台のタブレットで対面プレイができます。

## 概要

16個のコマには「色（明・暗）」「高さ（高・低）」「形（丸・四角）」「中心（穴あり・なし）」の4つの違いがあります。プレイヤーは **自分のコマを選べず、相手にコマを渡す** のがコア。たて・よこ・ななめの4マスで「共通する特徴が1つでも揃った」列を作った人の勝ちです。

## 操作方法

- **コマを選ぶ番**: 下の「のこっているコマ」から1つタップ → 相手に渡る
- **コマを置く番**: 盤面の空きマスをタップ → 渡されたコマが置かれる
- ヘッダーの「？ ルール」でいつでもあそびかたを確認できます

## ゲームルール

1. プレイヤー1が、プレイヤー2に渡すコマを1つ選ぶ
2. プレイヤー2はもらったコマを盤面の好きなマスに置く
3. 続けてプレイヤー2が、プレイヤー1に渡すコマを選ぶ
4. 以後、交互に「相手に渡す→受け取って置く」を繰り返す
5. 4つ並んだ列に「全員が共通する特徴」が1つでも揃ったら、置いた人の勝ち
6. 全16コマを置き切っても揃わなければ引き分け

## 構成

- `index.html`: 自己完結HTML+CSS。盤・パネル・ストック・モーダル・紙吹雪を含む
- `game.js`: 状態管理、勝利判定、レンダリング、効果音
- `README.md`: このファイル

## 状態管理

- `state.board`: 16マスの配置（コマ番号 0〜15 or null）
- `state.available`: 残コマのSet
- `state.selectedPiece`: 渡されたコマ
- `state.phase`: `'select'` / `'place'` / `'over'`
- `state.currentPlayer`: いま手番のプレイヤー（1 or 2）
- `state.winner`: `1` / `2` / `'draw'` / `null`
- `state.scores`: 通算勝ち数（リセットで0に戻る）

## コマのエンコード（4ビット）

| ビット | 1 | 0 |
|---|---|---|
| bit0 | 高い | 低い |
| bit1 | 暗い | 明るい |
| bit2 | 四角 | 丸 |
| bit3 | 穴あり | 詰まり |

勝利判定は4コマのANDとORを取って共通ビットを抽出。

## レスポンシブ対応

- 縦置きタブレット: P2上 / 盤 / ストック / P1下
- 横置きタブレット (≥900px): P2左 / 盤中央 / P1右 / ストック下
- 対面プレイのため P2 パネルは縦置き時180度回転
- スマホ幅でも遊べる（コマピッカーが小さくなる）

## 効果音

WebAudio APIで合成（外部ファイル不要）。

- `select`: 短い高音（ストックタップ）
- `handoff`: スライド音（コマを相手に渡す）
- `place`: 木のコツン音（盤に置く）
- `win`: 上昇トライアド
- `draw`: 下降2音
- `tap`: ボタンタップ音
- ヘッダーの 🔊 でON/OFF切り替え可
