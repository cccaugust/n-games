# ベリーバウンス（berry-bounce）

## 概要
ふってくるベリーをキャッチしてスコアをのばすゲームです。💣を取るとライフがへります。

## 操作方法
- **PC**: ← → で移動（Space で結果画面からすぐ再スタート）
- **スマホ/タブレット**: 画面をドラッグして移動

## ゲームルール
- ベリーを取ると **+1点**
- ベリーを取りそこねると **ライフ -1**
- 💣を取ると **ライフ -2**
- ライフが 0 で終了（スコア保存 & ランキング表示）

## 素材（重要）
- プレイヤー画像は `public/games/berry-bounce/assets/player.png`（URL: `/games/berry-bounce/assets/player.png`）を読み込みます
  - このファイルが無い/読めない場合は、ゲーム内で簡易キャラ（丸いフォールバック）が表示されます

## 構成
- `index.html`: 画面レイアウト（HUD/オーバーレイ）
- `style.css`: 見た目（カード/キャンバス/オーバーレイ）
- `game.js`: ゲーム本体（Canvas描画、入力、スコア保存）

## 状態/データ
- スコアは Supabase の `scores` テーブルへ `game_id = "berry-bounce"` として保存します
