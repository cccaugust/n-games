# Game: yojijukugo-quiz（四字熟語クイズ）

## 概要
小学生向けの四字熟語クイズです。出題（よみ/いみ）を見て、下の漢字パネルから4文字を選び、4つのマスに入れて正解を作ります。

## 操作方法
- **PC**: 漢字パネルをドラッグしてマスへドロップ / パネルをクリック→マスをクリックでもOK
- **タッチ**: 漢字パネルをドラッグしてマスへ / パネルをタップ→マスをタップでもOK

## ルール / 仕様
- **目的**: 出題（よみ/いみ）から正しい四字熟語を完成させる
- **出題形式**: `よみ` / `いみ` / `よみ+いみ` を切り替え可能（localStorageに保存）
- **パネル数**: 正解4文字 + ダミーを含めて合計10個程度
- **正解時**: スコア+1、連続正解+1、紙ふぶき演出 → 自動で次の問題へ
- **不正解時**: 連続正解は0、マスが揺れて再挑戦

## 画面 / レスポンシブ
- PC/タブレット/スマホ対応（`clamp()`で文字/マスサイズが自然に変化）
- `viewport-fit=cover` + `safe-area-inset` で端の食い込みを抑制
- タイルは `touch-action: none` でドラッグ中の誤スクロールを抑制

## 実装メモ（設計）
- **エントリ**: `index.html`
- **メインロジック**: `game.js`
  - `BANK`: 四字熟語データ（word/yomi/meaning）
  - `newRound()`: 出題と10枚パネル生成（正解4文字 + 他の熟語からダミー）
  - ドラッグ: Pointer Events で簡易D&D（モバイルでも動く）

## ファイル構成
- `index.html`: 画面UI（モード切替、問題表示、4マス、パネル一覧）
- `style.css`: ポップな見た目・アニメーション（揺れ/紙ふぶき/ハイライト）
- `game.js`: ゲーム進行、D&D、正誤判定、エフェクト
