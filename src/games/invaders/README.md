# インベーダー（Invaders）

下から撃つタイプのスタンダードなインベーダーゲームです。

## あそび方

- **PC**: ←→ / A D で移動、スペースで発射
- **スマホ**: 画面下のボタン（◀ ▶ / FIRE）

## ステージ

- `stage-select.html` でステージを選択してプレイできます
- デフォルトで **3つ** のステージが入っています
- `editor.html` で自分のステージを作れます

## ステージ作成（ドット絵メーカー連携）

ステージ作成画面では、ペイントソフトみたいにドラッグで敵キャラを配置できます。

- **デフォルト（👾）**: 内蔵の敵キャラ
- **ドット絵（キャラ）**: 本プロジェクトの「ドット絵メーカー」で作って**保存**した作品を、敵として選んで配置できます

※ 作ったステージはブラウザの `localStorage` に保存されます。
