# Game: aquarium

## 概要
水槽に魚を追加して、泳ぎ方（性格）や見た目（色相/サイズ）を編集できるゲーム（エディタ）です。  
魚の素材は **ドット絵メーカー（`PixelAsset`）** と共通で、ここで作った魚はドット絵メーカー側にも保存されます。  
サンプルは魚だけでなく、**えび/かに/たこ/いか/くらげ/ひとで/貝/うに** などの「魚介類」や、**タツノオトシゴ/マンタ/チョウチンアンコウ/ジンベエ** みたいな海の生き物も入っています。

## 操作
- **魚を追加**: 上のヘッダに並ぶ魚のアイコンを **ドラッグ&ドロップ** で水槽へ
  - スマホ/タブレットは、魚アイコンを **長押し→そのまま水槽へ**（擬似ドラッグ）
- **魚の名前を見る**: ヘッダの魚アイコンをタップ/クリック（短い表示）
- **魚を編集**: 水槽で泳いでいる魚をタップ/クリック → 編集ダイアログ
  - **変更できる**: 名前 / 性格 / 色相（Hue）
  - **削除**: ダイアログの「削除」
- **自作の魚を追加（描く）**: ヘッダ左の「＋」→ ドット絵エディタを開く

## 構成
- `index.html`: 画面UI（上:パレット / 下:水槽 / 魚編集ダイアログ）
- `game.js`: Phaserシーン・魚ロジック・パレット/ドラッグ/ダイアログ連携
- `style.css`: レイアウト（全画面水槽）とUI部品

## データ
- 魚素材は `src/js/pixelAssets.js` の `PixelAsset` として保存（IndexedDB + Supabase）。
- 水槽の「魚の配置/設定」は現状はメモリ上（将来 localStorage / Supabase 保存に拡張予定）。

## 画面 / レスポンシブ
- PC/スマホ/タブレット: **画面全体が水槽**（上にパレット、下は水槽で固定）
- 縦/横回転: Phaser は `RESIZE` で追従（トップバー高さはCSS変数で反映）
