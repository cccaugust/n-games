# Game: aquarium

## 概要
水槽に魚を追加して、泳ぎ方（性格）や見た目（色相/サイズ）を編集できるゲーム（エディタ）です。  
魚の素材は **ドット絵メーカー（`PixelAsset`）** と共通で、ここで作った魚はドット絵メーカー側にも保存されます。

## 操作
- **魚を追加**: 右上の「＋ サンプル」「＋ 作品」「＋ 作る」
- **魚を選ぶ**: 右の魚リストをタップ/クリック
- **設定変更**: 名前 / 性格 / 速さ / 大きさ / 色相
- **削除**: 選んだ魚の「削除」

## 構成
- `index.html`: 画面UI（左:水槽 / 右:編集UI）
- `game.js`: Phaserシーン・魚ロジック・UI連携
- `style.css`: レイアウトとモーダル

## データ
- 魚素材は `src/js/pixelAssets.js` の `PixelAsset` として保存（IndexedDB + Supabase）。
- 水槽の「魚の配置/設定」は現状はメモリ上（将来 localStorage / Supabase 保存に拡張予定）。
