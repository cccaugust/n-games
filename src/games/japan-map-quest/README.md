# 日本探検クエスト

## 概要
日本列島を探検しながら、47都道府県の特産物を集めるゲームです。
地図上のマーカーをタップして特産物をゲットし、コンプリートを目指します。

**完全無料・APIキー不要！** OpenStreetMap + Leafletを使用しています。

## 遊び方

### 操作方法
- **地図**: ドラッグで移動、ピンチ（またはスクロール）でズーム
- **マーカー**: タップするとポップアップが開く → 「ゲットする！」ボタンで収集

### ボタン
- **📍 場所を見る**: ミッションの都道府県へ移動
- **スキップ**: 別のミッションに変更
- **📦 コレクション**: 収集状況を確認（収集済みをタップするとその場所へ移動）
- **🗾 全体表示**: 日本全体が見えるビューに戻る
- **🔄 リセット**: 収集状況をリセット

## ルール
- **ミッション**: 画面下に表示される都道府県の特産物を探してゲット
- **クリア**: 47都道府県すべての特産物を集めるとコンプリート
- **クリアタイム**: プレイ開始からの経過時間を記録

## ファイル構成
- `index.html`: 画面レイアウト（タイトル画面 + ゲーム画面）
- `style.css`: レスポンシブUI、カスタムマーカースタイル
- `game.js`: Leaflet連携、収集システム、ミッション

## 技術仕様
- **地図**: Leaflet + OpenStreetMap（無料・APIキー不要）
- **データ保存**: localStorage
  - `japan-quest-collected`: 収集済みID配列
  - `japan-quest-start-time`: プレイ開始時刻

## 対応デバイス
- PC（Chrome, Firefox, Safari, Edge）
- タブレット
- スマートフォン
