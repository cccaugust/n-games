# 日本探検クエスト

## 概要
Google Mapsで日本列島を探検しながら、47都道府県の特産物を集めるゲームです。
地図上のマーカーをタップして特産物をゲットし、コンプリートを目指します。

## 遊び方

### 起動時
1. **Google Mapsモード**: Google Maps API Keyを入力してスタート
2. **デモモード**: APIキーなしで遊べる簡易版

### 操作方法
- **地図**: ドラッグで移動、ピンチでズーム
- **マーカー**: タップで特産物をゲット
- **ストリートビュー**: 「歩く」ボタンで現在地のストリートビューを表示（Google Mapsモードのみ）

## ルール
- **ミッション**: 画面下に表示される都道府県の特産物を探してゲット
- **コレクション**: 集めた特産物は「コレクション」画面で確認
- **クリア**: 47都道府県すべての特産物を集めるとコンプリート

## ファイル構成
- `index.html`: 画面レイアウト（API設定画面 + ゲーム画面）
- `style.css`: レスポンシブUI
- `game.js`: Google Maps連携、デモモード、収集システム、ミッション

## データ保存
- **APIキー**: localStorage（`japan-quest-api-key`）
- **収集状況**: localStorage（`japan-quest-collected`）

## 対応デバイス
- PC（Chrome, Firefox, Safari, Edge）
- タブレット
- スマートフォン

## Google Maps API の設定
1. [Google Cloud Console](https://console.cloud.google.com/) にアクセス
2. プロジェクトを作成
3. Maps JavaScript API を有効化
4. APIキーを作成
5. ゲーム起動時にAPIキーを入力

## 技術仕様
- Google Maps JavaScript API
- ストリートビュー対応
- レスポンシブデザイン（100svh使用）
- セーフエリア対応（iOS）
