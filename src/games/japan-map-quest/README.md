# Game: japan-map-quest（日本マップクエスト）

## 概要
日本列島っぽいフィールドを自由に歩き回り、表示されるミッション（例: 「奈良県へ行け！」）を達成するゲームです。  
海に出ると船の見た目になり、ミニマップで位置を確認できます。

## 操作方法
- **PC**: 矢印キー / WASD で移動
- **タッチ**: 画面下のDパッドで移動（押しっぱなし可）

## ルール / 仕様
- **目的**: ミッションで指定された県に到達する
- **クリア**: いまいる県が目的県と一致したらクリア（次のミッションが自動で選ばれる）
- **海移動**: 可能（海の上では船になります）

## 画面 / UI
- **HUD**: 「いま何県」「地形（陸/海）」「ミッション」
- **ミニマップ**: **256×256のドット**で日本列島のシルエット + プレイヤー位置 + 目的地マーカー
- **ボタン**: ミッション変更 / 東京にもどる

## 実装メモ（設計）
- **県判定**: 現状は「ざっくり座標（バウンディングボックス）」です  
  精度アップしたい場合は、以下のどちらかに差し替える想定です:
  - タイルごとの県IDマップ（2D配列）
  - GeoJSON + point-in-polygon（投影含む）

## ファイル構成
- `index.html`: 画面レイアウト（canvas + HUD）
- `style.css`: レスポンシブUI（スマホはDパッド表示）
- `game.js`: マップ生成、移動、県判定、ミニマップ描画、ミッション
