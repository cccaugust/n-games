# それはオレの魚だ！（Hey, That's My Fish!）

2〜4人の対面プレイで遊べるヘックス陣取りゲーム。1台のタブレットを囲んで、ペンギンで魚を取り合います。

## 概要

氷の盤面（六角タイル60枚）にペンギンを置き、順番にまっすぐ動かして魚を集めます。動いたあとの氷タイルは海に沈むので、盤面はどんどん狭くなり、ペンギンは身動きが取れなくなっていきます。最後にいちばん多く魚を集めた人が勝ち。

## 操作方法

### 配置フェーズ
- 自分の番に、光っている **魚1匹のタイル** をタップ → ペンギンが配置される
- 全員が全ペンギンを置き終えると、自動でゲームフェーズへ

### ゲームフェーズ
1. 自分のペンギン（光っている）をタップ
2. 移動できるマス（黄色いハイライト）が表示される
3. 行きたいマスをタップ → ペンギン移動 + 元タイルが沈む
4. 「選択をやめる」で選び直せる

## ゲームルール

- ペンギンは **6方向のいずれかに直線で** 何マスでも進める
- 通り道に他ペンギンや沈んだタイルがあると そこまでで止まる
- 元いたタイルは海に沈み、そのタイルの魚数があなたのスコアに
- 全プレイヤーのペンギンが1匹も動けなくなったら終了
- 終了時、まだペンギンが乗ってるタイルの魚も最後にスコア加算
- 魚の合計が多い人の勝ち（同点は引き分け）

## 人数とペンギン数

| 人数 | 1人あたりのペンギン |
|------|-------------------|
| 2人  | 4匹               |
| 3人  | 3匹               |
| 4人  | 2匹               |

## 構成

- `index.html`: 自己完結HTML+CSS、SVG盤面の入れ物、モーダル類
- `game.js`: 状態管理、ヘックス座標計算、移動ロジック、レンダリング、効果音
- `README.md`: このファイル

## 状態管理

- `state.tiles`: Map<"q,r", { q, r, fish, removed }>（軸座標の六角タイル）
- `state.penguins`: { id, player, q, r, placed }
- `state.players`: { id, name, color, score, ... }
- `state.phase`: `'menu'` / `'placing'` / `'playing'` / `'over'`
- `state.currentPlayer`: いま手番のプレイヤーID
- `state.selectedPenguinId`: 選択中のペンギンID
- `state.movableTargets`: 移動可能マスのリスト
- `state.winner`: 勝者プレイヤーIDの配列（同点なら複数）

## 座標系

軸座標（Pointy-top hexagonal axial）。6方向の隣接：

```
[+1, 0], [+1, -1], [0, -1], [-1, 0], [-1, +1], [0, +1]
```

ピクセル変換：
- `x = R * √3 * (q + r/2)`
- `y = R * 1.5 * r`

## レスポンシブ対応

- 縦置きタブレット: 上段(P2,P4)/盤/下段(P1,P3)、上段は180度回転
- 横置きタブレット: 同レイアウト、SVGがレスポンシブに伸縮
- スマホでもプレイ可能（フォントとペンギン絵文字が縮小）

## 効果音

WebAudio APIで合成（外部ファイル不要）。

- `select`: ペンギン選択音
- `place`: 配置時のポンッ音
- `move`: スライド音
- `eat`: 魚を獲得（魚数で音程変化）
- `invalid`: 不正タップ
- `win`: 勝利時の上昇トライアド
