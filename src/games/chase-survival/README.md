# 逃走サバイバル

ハンターから逃げ続けて賞金を稼ぐ3Dチェイスゲーム！

## 概要

「逃走中」をモチーフにした3Dアクションゲームです。フィールド内を逃げ回り、ハンターに捕まらないように賞金を稼ぎましょう。時間が経つほどハンターが増え、難易度が上がります。

## 操作方法

### PC
- **WASD / 矢印キー**: 移動
- **Shift**: ダッシュ
- **マウス**: 視点操作（クリックでポインターロック）
- **ESC**: 一時停止

### タブレット / スマホ
- **左側ジョイスティック**: 移動（向きも自動で変わる）
- **右側ダッシュボタン**: ダッシュ

## ゲームルール

### 目標
- ハンターから逃げ続ける
- 長く生き延びるほど賞金が増える

### 難易度
| 難易度 | ハンター速度 | 追加間隔 | 最大ハンター数 | 秒あたり賞金 |
|--------|-------------|----------|---------------|-------------|
| かんたん | 遅い | 45秒 | 4体 | ¥1,000 |
| ふつう | 普通 | 30秒 | 6体 | ¥2,000 |
| むずかしい | 速い | 20秒 | 10体 | ¥5,000 |

### アイテム
フィールド上に出現するアイテムを取ると特殊効果が発動！

| アイテム | 効果 | 持続時間 |
|----------|------|---------|
| ⚡ スピードアップ | 移動速度1.5倍 | 5秒 |
| 🛡️ シールド | ハンターに触れても捕まらない | 5秒 |
| 👻 透明化 | ハンターに追跡されなくなる | 4秒 |
| ❄️ ハンター凍結 | 全ハンターが一時停止 | 3秒 |
| 💰 ボーナス | ¥10,000獲得 | - |

### ダッシュ
- ダッシュ中は2倍の速度で移動
- 1.5秒間持続
- 使用後5秒のクールダウン

## 技術的な詳細

### 使用ライブラリ
- **Three.js**: 3Dレンダリング

### 主要システム
- **3Dフィールド**: 100×100ユニットのグリッドフィールド
- **カメラ**: 3人称視点（プレイヤーの後方から追従）
- **ハンターAI**:
  - 追跡モード: プレイヤーを直接追う
  - パトロールモード: ランダムに徘徊
  - 障害物回避機能あり
- **衝突判定**: 球体ベースの当たり判定

### 障害物
- ビル（複数サイズ、窓の発光演出付き）
- 木
- コンテナ/ボックス
- フィールド境界の壁

## 画面 / レスポンシブ

- タブレット最適化（主要プレイ環境）
- スマホ/PC対応
- タッチ操作: バーチャルジョイスティック + ダッシュボタン
- PC: キーボード + マウス
- セーフエリア対応（env(safe-area-inset-*)）

## 効果音

Web Audio APIを使用した合成音：
- アイテム取得音
- ダッシュ音
- ハンター追加警告音
- ゲームオーバー音
