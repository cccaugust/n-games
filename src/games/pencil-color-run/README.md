# えんぴつカラーダッシュ

3Dのランゲームです。えんぴつが自動で進み、左右移動だけで遊べます。

## ルール
- 同じ色の色鉛筆を取ると長さアップ
- 違う色を取るとゲームオーバー
- 壁にぶつかると壁の高さ分だけ長さダウン
- 壁の高さより短いとゲームオーバー
- 半透明の色ゲートでえんぴつの色が変化
- 通った地面はえんぴつ色で塗られる

## 操作
- PC: `←` `→` / `A` `D`
- スマホ・タブレット: 画面下の左右ボタン

## 構成
- `index.html`: HUD、ルール表示、タッチUI
- `game.js`: Three.jsのシーン、生成、当たり判定、演出
