# PAINT WARS

スプラトゥーン風3D見下ろしシューティングゲーム。制限時間内にマップをより多く自分の色で塗り、CPUチームに勝利せよ！

## 概要

- **試合時間**: 3分
- **勝利条件**: 終了時に自チームの塗り面積が多い方の勝ち
- **チーム構成**: プレイヤー1人 + 味方CPU2人 vs 敵CPU3人
- **リスポーン**: 倒されると3秒後に自陣スタート地点で復活

## 操作方法

### PC（キーボード＋マウス）

| 操作 | キー |
|------|------|
| 移動 | W/A/S/D |
| 照準 | マウス |
| メイン攻撃 | 左クリック |
| サブウェポン | 右クリック |
| スペシャル | Qキー |
| スライド（デュアル専用） | Shiftキー |
| ポーズ | Escキー |

### モバイル（タッチ）

- 左ジョイスティック: 移動
- 右ジョイスティック: 照準
- 射撃ボタン: メイン攻撃
- サブボタン: サブウェポン
- スペシャルボタン: スペシャルウェポン

## ゲームメカニクス

### インクシステム

- **自分のインクの上**: 移動速度UP + インク回復3倍
- **敵のインクの上**: 移動速度DOWN
- **インク回復**: 時間経過で自動回復（10%/秒）

### 武器一覧

| 武器 | 特徴 |
|------|------|
| シューター | バランス型。扱いやすい基本武器 |
| ローラー | 塗り特化。転がして広範囲を塗る |
| チャージャー | 狙撃型。長押しチャージで高威力 |
| ブラスター | 範囲攻撃。着弾点で爆発 |
| デュアル | 機動型。スライド回避が可能 |

### サブウェポン

| サブ | 効果 | クールダウン |
|------|------|------|
| スプラッシュボム | 投げて爆発、範囲ダメージ | 8秒 |
| シールド | 3秒間ダメージ無効 | 15秒 |
| センサー | 敵の位置を表示 | 12秒 |

## キャラクター

6種類のキャラクターから選択可能（見た目のみの違い、性能差なし）

- イカボーイ
- イカガール
- タコスナイパー
- ローラーキッズ
- ヘビーガンナー
- ニンジャ
