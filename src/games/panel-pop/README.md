# Panel Pop

「パネルでポン」風のリアルタイム対戦パズルを、1人用スコアアタックとして再構築したゲームです。

## 遊び方

- ←→↑↓: カーソル移動
- Space: 2つのパネルを入れ替え
- Shift: せり上げ加速
- 3つ以上同色を縦か横にそろえると消えます
- 連鎖が続くと CHAIN が増えて高得点になります

## ルール

- 時間経過で盤面がせり上がります
- パネルが上部の隠し行に到達するとゲームオーバー
- スコアに応じてレベルが上がり、せり上げ速度が上昇

## 実装メモ

- Canvas描画のため軽量で高速
- マッチ検出は縦横ランレングスで実装
- クリア猶予 (`clearTimer`) と `chainWindow` で連鎖判定
