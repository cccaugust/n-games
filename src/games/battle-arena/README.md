# バトルアリーナ

スマブラ風の「ふっとばし」2D対戦アクション（ローカル2人対戦）のプロトタイプです。

## 操作

### PC（キーボード）
- **P1**
  - 移動: `A` / `D`
  - ジャンプ: `W`（2段ジャンプ）
  - こうげき: `F`
  - つよい: `G`
- **P2**
  - 移動: `←` / `→`
  - ジャンプ: `↑`（2段ジャンプ）
  - こうげき: `K`
  - つよい: `L`
- **共通**
  - もう一回: `R`

### スマホ/タブレット（タッチ）
画面下のボタンで操作できます（左右/ジャンプ/こうげき/つよい）。

## ルール
- こうげきを当てると相手の **%** が増え、**% が大きいほど強くふっとぶ** ようになります。
- 場外に落ちると **ストックが 1 減ります**。
- 先にストックが 0 になった方が負けです（初期ストックは 3）。

## 構成
- `index.html`: 画面レイアウト（戻る、HUD、操作説明、タッチボタン）
- `style.css`: 画面/ボタンの見た目、セーフエリア対応
- `game.js`: Phaser のゲーム本体（ステージ/キャラ/当たり判定/%/復活）

## 状態/データ
- 永続化（localStorage/Supabase）は **未使用**。
- 1ラウンド内の状態は Phaser の `BattleScene` 内で管理。

## 画面/レスポンシブ
- PC/タブレット/スマホ対応（スマホはタッチボタンを表示）
- セーフエリア（`env(safe-area-inset-*)`）を考慮
- ゲーム画面は 16:9 の比率でフィット表示（Phaser Scale FIT）
