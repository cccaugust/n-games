# ボイスビートラボ

こえやおとを **1秒だけサンプリング**して、8ステップのシーケンサーで「ヘンテコ曲」を作るミニゲームです。

## あそびかた（操作）

- **マイクをつかう**: 「🎤 マイクをつかう」を押してマイク許可
- **録音**: A〜Dの「● とる！」で **1.0秒** 録音（自動で止まる）
- **再生**: 「▶︎ ならす」や、パッドの空白部分をタップで再生
- **曲にする**: 8つのマスをタップでON/OFF → 「▶︎ スタート」
- **テンポ**: スライダーでBPM変更
- **ヘンテコモード**: ランダムで「チップモンク/ロボ/ぎゃくさいせい」っぽくなる
- **ぎゃはは！**: ちょっとカオスに鳴るボタン（ネタ用）

## ルール（スコアなど）

- スコアはありません（作って遊ぶタイプ）
- 録音データはページを閉じると消えます（保存なし）

## 構成

- `index.html`: UI（パッド / シーケンサー / ヘルプ）
- `style.css`: レイアウトとアニメーション（レスポンシブ含む）
- `game.js`: 録音（MediaRecorder）/ 再生（Web Audio API）/ シーケンサー

## 画面/レスポンシブ

- PC/タブレット/スマホで動く前提（大きめボタン、2カラム→1カラム切り替え）
- `viewport-fit=cover` + safe-area を考慮した余白

## 既知の制約

- 録音は **MediaRecorder** を使うので、ブラウザによっては非対応のことがあります
- マイクは **HTTPS** またはローカル開発環境で動きます
