# ユニティキャッチ（unity-catch）

落ちてくるコインをキャッチするミニゲームです。  
このゲームは **Unityの無料アセット（PNGなど）を差し替えできる** ように作ってあります。

## 遊び方
- ドラッグ（指/マウス） or `←` `→` で左右に移動
- 🪙コイン: +1点
- 💎ジェム: +5点
- 💣ボム: ライフ-1
- ❤️ハート: ライフ+1（最大5）

## Unity素材を組み込む（差し替え）
Unity側で使える無料素材のうち、**テクスチャ（PNG）** を書き出して以下に置くと、ゲーム内で自動的に読み込まれます。

置き場所（このリポジトリ）:
- `public/games/unity-catch/assets/`

ファイル名（この名前で置く）:
- `player.png`（自キャラ/かご）
- `coin.png`
- `gem.png`
- `bomb.png`
- `heart.png`
- `bg.png`（背景、任意）

※ 画像が見つからない場合でも、フォールバックの図形描画で動作します。

## スコア
ログインしている場合、スコアがSupabaseに保存され、ランキングが表示されます。
