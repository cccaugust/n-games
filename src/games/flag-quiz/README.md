# 国旗あてクイズ

## 概要
国旗を見て、国の名前を4択で当てるクイズです。正解するとキラッ＆紙ふぶき！

## 操作方法
- **タップ/クリック**: 4つの選択肢から答えを選ぶ
- **キーボード**: `1`〜`4` で選ぶ

## ゲームルール
- **10もん**: 10問で終了（結果画面にスコア表示）
- **エンドレス**: ずっと続く（間違えても続行）
- **スコア**: 正解で +100、連続正解でボーナス（れんぞくが増えるほど少し加点）

## 構成
- `index.html`: 画面（UIの土台）
- `style.css`: 見た目（レスポンシブ/演出）
- `game.js`: 出題ロジック、スコア、正解/不正解エフェクト、SE

## 状態/データ
- `localStorage` に保存:
  - `flagQuiz.mode`: モード（`ten` / `endless`）
  - `flagQuiz.se`: 効果音ON/OFF
  - `flagQuiz.best`: ベストスコア

## 画面/レスポンシブ
- PC/タブレット/スマホ対応（ボタンは最小でも押しやすいサイズ）
- `viewport-fit=cover` と `safe-area-inset` を考慮
- スマホ幅では選択肢が **1列** に自動切り替え

## 注意点
- 国旗画像は **FlagCDN**（オンライン）から読み込みます。
  - オフライン/ブロック環境では画像が出ない場合があり、そのときは絵文字表示になります。
