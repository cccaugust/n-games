# OTRIO

3x3の盤に「小・中・大」の輪を置いてそろえる、2人対戦ボードゲームです。タブレットで遊びやすいように大きいタップUIで実装しています。

## 操作
- 画面下のサイズボタン（小/中/大）を選ぶ
- 盤面のマスをタップして配置
- リセットで最初からやり直し

## 勝利条件
- 同じサイズの輪を縦・横・斜めで3つそろえる
- 同じマスに小中大を同じプレイヤーでそろえる
- 直線上で「小→中→大」または「大→中→小」をそろえる

## 構成
- `index.html`: UI（盤面/状態表示/操作ボタン）
- `game.js`: ルール、手番、勝敗判定

## レスポンシブ
- `viewport-fit=cover` と safe-area inset 対応
- 盤面は `min(82vmin, 680px)` でタブレット最適化
- ボタンは最小44px以上
