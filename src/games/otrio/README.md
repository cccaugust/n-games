# OTRIO

3x3の盤に「小・中・大」の輪を置いてそろえる、最大4人対戦のボードゲームです。タブレット/スマホ/PCで遊びやすいよう大きめのタップUIにしています。

## 操作
- スタート画面で人数（2〜4人）を選んで「はじめる」
- 画面下のサイズボタン（小/中/大）を選ぶ → 盤面のマスをタップで配置
- マスをホバー（PC）すると半透明の **ゴーストリング** で置いた結果がプレビューされます
- 右上の 🔊 ボタンで効果音 ON/OFF
- 「リセット」で同じ人数のまま再開、「新しい対戦」で人数選び直し

## プレイヤー
- 最大4人まで参加可
- カラー: P1 青 🐬 / P2 ピンク 🌸 / P3 緑 🌿 / P4 黄 ⭐
- 各プレイヤーは小×3 / 中×3 / 大×3 を持ちスタート

## 勝利条件
- 同じサイズの輪を縦・横・斜めで3つそろえる
- 同じマスに 小・中・大 を全部自分のリングでそろえる（入れ子）
- 直線上で「小→中→大」または「大→中→小」とサイズ階段で並べる

置けないプレイヤー（在庫切れ・置き場なし）の番は自動でスキップされます。誰も置けなくなったら引き分け。

## エフェクト・サウンド
- リング配置時にプレイヤーカラーのパーティクル＆リングパルス
- 勝利時に紙吹雪 + 勝利アルペジオ
- 効果音は **Web Audio API** で生成（音声ファイル不要）

## 構成
- `index.html`: UI（セットアップオーバーレイ、プレイヤーチップ、盤面、ピッカー、リザルト）
- `game.js`: 状態管理、ルール、勝敗判定、サウンド（SFX）、パーティクル、紙吹雪

## 状態 / データ
- ローカル状態のみ（永続化なし）
- 主要オブジェクト: `state.numPlayers`、`state.board`（9マス × {s,m,l}）、`state.stock`、`state.current`、`state.selectedSize`

## レスポンシブ
- `viewport-fit=cover` と safe-area inset 対応
- 盤面は `min(82vmin, 560px)` でタブレット最適化、`100svh` 想定
- ボタンは最小44px以上、タッチ向けに大きめ
- プレイヤーチップは `auto-fit, minmax(140px, 1fr)` で自動折返し

## 既知の課題
- AI（CPU相手）モード未実装。現状はローカル多人数対戦のみ
