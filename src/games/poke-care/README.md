# Game: poke-care

## 概要
ポケモンを選んで「えさ / あそぶ / ねる / なでる」でお世話し、レベルアップを目指すミニ育成ゲームです。

## 操作方法
- **PC**: クリック（なでる/ボタン）
- **タッチ**: タップ（なでる/ボタン）、選択画面はスワイプでページ切り替えもOK

## ルール / 仕様
- **目的**: お世話して「なかよしレベル」を上げる
- **スコア**: お世話ポイント（XP）。行動で増え、一定値でレベルアップ
- **クリア条件**: Lv.5 でクリア表示（そのまま続行も可能）

## 画面 / UI
- **選択画面**: パートナー（ポケモン）を選ぶ。◀▶ボタン/スワイプでページ切り替え
- **メイン画面**: ステータス（おなか/きげん/げんき）、XPバー、行動ボタン（えさ/あそぶ/ねる）
- **操作ヒント**: 「タップでなでる」やトースト表示で次にやることが分かるように表示

## 実装メモ（設計）
- **エントリ**: `index.html`
- **メインロジック**: `game.js`（Phaser設定）/ `scenes/*.js`（各画面）
- **主要な状態(state)**: `stats`（hunger/happiness/energy/xp/level）、`pokemonId`
- **データ保存**: `localStorage`（`poke-care-id`, `poke-care-stats`）

## ファイル構成
- `index.html`: 画面と簡易UI（もどるリンク/ヒント）
- `game.js`: Phaserゲーム起動設定
- `scenes/BootScene.js`: アセット読み込み・初期遷移
- `scenes/SelectionScene.js`: パートナー選択（ページ切り替え）
- `scenes/MainScene.js`: お世話/ステータス/レベルアップ
