# unity-catch assets

このフォルダは **Unityの無料アセット（PNGなど）を置く場所** です。  
`src/games/unity-catch/game.js` が、ここにある画像を次の固定パスで読み込みます:

- `/games/unity-catch/assets/player.png`
- `/games/unity-catch/assets/coin.png`
- `/games/unity-catch/assets/gem.png`
- `/games/unity-catch/assets/bomb.png`
- `/games/unity-catch/assets/heart.png`
- `/games/unity-catch/assets/bg.png`（任意）

## 使い方
1. Unity（またはAsset Storeの配布物）から、使いたいスプライト/テクスチャを **PNG** として用意
2. それぞれ上のファイル名に合わせてこのフォルダに配置
3. ブラウザでゲームを開くと自動で差し替わります

## 注意（ライセンス）
「無料」でも再配布やクレジット表記の条件があることがあります。  
素材のライセンス（利用範囲/表記/再配布可否）は必ず確認してください。
